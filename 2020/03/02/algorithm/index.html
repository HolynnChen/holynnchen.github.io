最长递增子序列

发表于 2020-03-02 阅读次数：

前言：看了别人一道算法题思考了几分钟才想明白解法，老了老了，好久没碰算法了，重新记录下

题目

  在一串非排序数组中，寻找一最长递增子序列，并返回其长度。

解法

O(n2)

  使用动态规划解法去做。

  当为第一个时，必为0。为避免max([])的情况出现，增加了一个[1]到原数组。+1代表加上位置i上的数字后子序列长度增加1。

O(nlog(n))

  寻找贪心情况下递减子序列个数

  详细解答可以看这个，这里主要记录一下实现。有一个重要的点就是，tops必然是递增的。
